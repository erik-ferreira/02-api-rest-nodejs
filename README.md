# RF - Requisitos Funcionais

- [ ] o usuário deve poder criar uma nova transação;
- [ ] o usuário deve poder obter um resumo da sua conta;
- [ ] o usuário deve poder listar todas as transações que já ocorreram;
- [ ] o usuário deve poder visualizar uma única transação;

# RN - Regras de Negócio

- [ ] A transação pode ser do tipo crédito que somará ao valor total ou débito que subtrairá do valor total;
- [ ] Deve ser possível identificar o usuário entre as transações;
- [ ] O usuário só pode visualizar as transações o qual ele criou;
